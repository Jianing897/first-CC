### 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| `summary` | 是 | 长度必须为 **10** |
| `articles` | 是 | 长度必须为 **10** |
| `source_urls` | 是 | 每项含 `name` + `url`；合并条可多项 |
| `paragraphs` | 是 | 合并后总汉字 ≤500 |
| `topics` | 建议 | 见 `coverage_policy.md` 主题代码 |
| `char_count` | 建议 | 正文字数自检 |

`source` 单字段已废弃，统一用 `source_urls`。

## 选题流程（Agent）

1. 读取 `raw/items.json`，丢弃 `likely_analysis: true` 及国别不在清单者
2. 按国别分组，组内按日期排序
3. 每组写成一条（合并），优先覆盖 `coverage_policy.md` 七类主题
4. 从候选国别中选出 **10 国**，区域尽量均衡
5. 生成 `structured.json`，运行 `build_docx.py` 前校验条数与字数
