# 选题与覆盖范围（研究室口径）

## 一、关注主题（国别为主体）

每条须可归入以下至少一类，且以**该国**油气投资/投资环境为叙述主体：

| 代码 | 主题 | 典型事件 |
|------|------|----------|
| `regime` | 油气管理体制 | 机构改革、许可制度调整、国家石油公司权责变化 |
| `fiscal` | 财税 | 税率、暴利税、关税、补贴、利润分成调整 |
| `tender` | 招标 | 区块招标、服务招标、招投标结果、授标 |
| `discovery` | 重要发现 | 新油田/气田发现、试油成功、储量认定 |
| `energy_plan` | 油气能源计划 | 产量目标、扩产/减产决定、国家能源规划落地 |
| `carbon` | 碳政策 | 碳税、ETS、甲烷管控、CCUS 法规、绿电绑定要求 |
| `geopolitical` | 地缘政治事件 | 制裁、袭击、封锁、出口禁令、外交冲突直接影响油气 |

**不收录**：纯市场评论、油价预测、机构展望、情景分析、「或」「可能」「预计」类前瞻判断。

## 二、事实 vs 分析/预测（硬过滤）

### 排除（标题或摘要命中即标记 `likely_analysis`）

英文：`forecast`, `predict`, `outlook`, `expects`, `expected`, `could hit`, `could reach`, `sees`, `analyst`, `analysis`, `estimate`, `projected`, `projection`, `scenario`, `warning`, `bullish`, `bearish`, `strategy`, `recommend`, `likely to`, `may hit`, `if ceasefire`, `if conflict`

中文：`预计`, `预测`, `展望`, `分析认为`, `机构认为`, `或将`, `有望`, `可能达`, `或突破`, `若…则`, `情景`, `策略`, `建议`

### 保留（已发生事件信号）

`签署`, `授予`, `批准`, `发现`, `中标`, `招标`, `袭击`, `停产`, `复产`, `制裁`, `通过法案`, `上调税率`, `下调税率`, `投产`, `投运`, `关闭`, `重启`, `发现油田`, `signed`, `awarded`, `approved`, `discovery`, `sanction`, `attack`, `outage`, `resumes`, `tender`, `bid`, `grants`, `passed`, `law`, `regulation`

成稿阶段：仅使用 `likely_analysis: false` 且能核实日期的素材。

## 三、重点区域（国别清单）

每期正文**10 条**，每条对应**一个国家**（同一国多条素材须合并为一条）。

| 区域 | 国别（正文用简称） |
|------|-------------------|
| 北美 | 美国 |
| 中东 | 伊拉克、阿联酋、卡塔尔、阿曼 |
| 中亚-俄罗斯 | 哈萨克斯坦、阿塞拜疆、俄罗斯 |
| 拉美 | 巴西、圭亚那、特立尼达和多巴哥、苏里南、阿根廷、墨西哥 |
| 亚太 | 印尼、马来西亚 |
| 非洲 | 阿尔及利亚、加蓬、纳米比亚、莫桑比克 |

**国别合并规则**：双周内同一国有多条已发生事件 → 合并为一条，标题概括主线，正文按时间或主题串联，**总字数 ≤500 字**。

**区域平衡**：10 条应尽量覆盖多个区域，避免单一区域过度集中（除非当期无其他国别素材）。

## 四、篇幅与体裁

| 项目 | 要求 |
|------|------|
| 每期条数 | **10 条**（摘要 10 条 + 正文 10 条，一一对应） |
| 单条字数 | 正文合计 **≤500 汉字**（不含标题与来源行） |
| 体裁 | **新闻时报体**：时间 + 主体 + 事实 + 关键数据，短句，少议论 |
| 来源 | 每条附 **具体 URL**（合并条可列多个链接） |

## 五、成稿合并示例

素材：

- 巴西：Petrobras 收购 Campos 区块 50% 权益
- 巴西：授予 Subsea7 深海合同

合并输出一条：

- 标题：`巴西：Petrobras 扩区块权益并授予大型深海合同`
- 正文：两段事实叙述，≤500 字
- 来源：`（信息来源：World Oil https://...；World Oil https://...）`
